# ASTRAL WARDEN（仮）

タイミングアクションRPGの **戦闘プロトタイプ**。
「腕次第で格上にも勝てる」テクニカルな戦闘の面白さを核に据えた、完全オリジナルのゲームです。
名称・キャラクター・用語・素材はすべて独自に作成しており、既存作品の固有名詞・アセットは一切使用していません。

技術スタック：**TypeScript + HTML5 Canvas + Vite**（戦闘はCanvas描画、メニュー類はDOM）。

---

## 戦闘システム

| システム | 内容 |
| --- | --- |
| **EN（エナジー）** | スキルで消費（コストはスキルごと）。**自動回復はしない**。回復は「休憩」と「ガード成功」時のみ |
| **武器とスキル** | 攻撃力・会心・Break値・レアリティは**武器本体**が持つ。スキルは「やり方」（ヒット数・対象数・会心・Break倍率・ためる/集中）を決め、攻撃力は持たない |
| **スキル種類** | ⚔攻撃（ヒット数・対象数・会心率/倍率・Break倍率）/ ▲ためる（次の攻撃2.3倍）/ ◎集中（次の行動EN0・自身0コスト） |
| **スキル連携** | スキルを特定の順番で出すと**追撃**が発生（ためる→攻撃／集中→攻撃／別系統の攻撃を連続）。1手目を出すと、連携になる次のスキルを持つ武器カードが**金色に光って通知**（`COMBOS`＝`src/game/data.ts`） |
| **会心** | 武器の会心率＋スキルの会心率で発生。会心ダメージは1.5倍（「会心2倍」等のスキルで倍率変化） |
| **武器相性** | 敵の種別ごとに弱点武器があり、弱点ヒットでダメージ増加（甲殻種=打撃 / 霊体種=斬撃 / 飛翔種=刺突） |
| **攻撃カウント** | 各敵の頭上に攻撃までのカウントを表示。**プレイヤーが行動するたびに1減り**、0になると予兆「!」が出て少し溜めてから攻撃（この間がガードタイミング） |
| **タイミングガード** | GUARD（軽減0.55＋EN小・地味）/ JUST（軽減0.25＋EN中・中間）/ **PERFECT**（無効化＋HP/EN大回復＋ヒットストップ＋弾き演出＋敵を怯ませ） |
| **ブレイク（気絶）** | 攻撃でゲージ蓄積→満タンで**3ターン**スタン。全ヒット会心（被ダメ増）、敵は攻撃不能。3ターン後に元の待ちへ復帰 |
| **複数敵** | タップ/`T`キーや左右矢印で対象選択。カウントが同時に0でも**左の敵から順番に**1体ずつ攻撃（必ずガードで対応可能） |

## 進行・収集システム

| システム | 内容 |
| --- | --- |
| **ウェーブ制** | 通常ステージ＝**3連戦**。HP/ENを引き継いで戦い抜き、**3戦目はボス**。途中で敗北すると入手はなし |
| **無限の回廊** | 終わりのないダンジョン。**階ごとにランダムな敵**（深いほど強化、5階ごとにボス）。1階クリアごとに報酬を確定し、**最高到達階を記録**。倒れるまで潜り続ける（`endlessFloorEnemies`＝`src/game/data.ts`） |
| **武器レアリティ** | コモン/アンコモン/レア/エピック/レジェンド/アストラル（虹）。**武器ごとに固定**。スキル数を決める（C/U/R=1、E/L/A=2）。エピック以上はインベントリ/ショップで枠が発光 |
| **ドロップ** | モンスターごとに「ドロップ武器＋ドロップ率」を設定。撃破時に確率で入手し、**武器レアリティ以下のプールからスキルを抽選**（低レアほど出やすい） |
| **ゴールド** | 敵を倒すと入手（**HP・攻撃力が高い／ボスほど多い**＝`rollEnemyGold`）。クリアで確定し、ショップで使用 |
| **ショップ：武器** | **ゴールドで武器を購入**（`SHOP_ITEMS`＝`src/game/data.ts`）。購入時もレアリティに応じてスキルを抽選。**一度買った武器は SOLD OUT**（購入履歴をセーブ） |
| **ショップ：宝箱** | **宝箱はゴールドで何度でも購入でき、購入＝即開封**（`SHOP_CHESTS`）。レアリティ帯ごとに価格が異なり、その帯の武器がランダムで出る（ガチャ的）。開封はモーダルで演出 |
| **武器強化（鍛冶屋）** | 武器に**レベル・経験値**を実装。下部ナビ「🔨鍛冶屋」から、強化対象→素材武器を選んで**経験値を与えてレベルアップ**。素材は装備中・ロック中以外を消費。必要経験値は**レアリティで増加**（`expForNext`／`materialExp`）。レベル/覚醒で攻撃・Break・会心が上昇（`effectiveWeapon`） |
| **覚醒（限界突破）** | **10レベルごとに「覚醒」の壁**。突破には**同じ武器を複数本合成**（必要数は覚醒回数で増加＝`awakenCost`）。覚醒でレベル上限が解放され（最大Lv60）、ステータスがさらに上昇 |
| **インベントリ** | 入手した武器を格納・付け替え。**レアリティでフィルタ／レア度で昇順・降順ソート／削除ロック**（ロック中は削除不可・「ロックのみ表示」）。武器ドット絵・レベル・覚醒を表示 |
| **セーブ** | localStorage に保存（インベントリ〔レベル/経験値/覚醒含む〕・装備・最深ステージ・ゴールド・購入履歴・ロック・無限の回廊の最高階） |

## 演出・フィードバック

| 要素 | 内容 |
| --- | --- |
| **攻撃／被弾モーション** | プレイヤー・敵ともに、攻撃時の踏み込み（専用フレーム）と被弾時ののけぞり＋フラッシュを表現 |
| **ダメージ表記** | 与ダメージを**赤橙のギザギザ爆発＋白縁取りの数字**で表示。会心/弱点/渾身はより大きく赤く＋修飾ラベル。約2秒残る |
| **味方ログ** | 回復・効果（JUST/集中/ためる/休憩など）は画面右下に**ログ的に積んで**表示 |
| **PERFECT演出** | ヒットストップ→ホワイトアウト→スロー＋弾きエフェクト＋画面シェイク |
| **撃破演出** | ノックバック＋フェード＋**コインが弾けて落ちる**＋レアリティ色の宝箱 |
| **宝箱開封** | リザルトで入手宝箱を並べ、**順番に開封アニメ**（揺れ→発光→中身がポップ）。**エピック以上は専用の派手な開封**（溜め発光＋光の粒子）。**アストラルは虹色の粒子＋虹リング**を盛大に表示 |
| **キャラアニメ** | 待機中の呼吸・浮遊、登場時のスライドイン、攻撃の踏み込みなどを手続き的に表現 |
| **ドット絵 / サウンド** | コード定義のオリジナルスプライト（キャラ・敵・武器・UIアイコン・宝箱）＋Web Audio合成のBGM・効果音（♪でミュート） |

## 画面・UI

全画面を**ライト基調・ターミナル風（ネオブルータリズム）**のデザインで統一。太い黒枠＋オフセット影、マゼンタ／シアンのアクセント、等幅フォントのラベルが特徴。

- **共通**：最上部に `BATTLE_ROYALE_V1` ＋所持ゴールド＋設定/ヘルプのバー、**画面下に常時固定**の **ホーム/ダンジョン/インベントリ/鍛冶屋/ショップ** ナビ（戦闘中は非表示・現在地はマゼンタ表示）
- **ホーム**：ロゴ＋「冒険に出る」
- **ダンジョン選択（DUNGEON HUB）**：ゾーンカード（ZONE番号・STATUS・敵サムネ・属性タグ・LOOT画像）をタップで選択 → 下部の **Power Comparison**（戦闘力 vs 推奨）と **ENTER DUNGEON**
- **ショップ（GRAND BAZAAR）**：**FLASH DEALS**（宝箱を横スクロールの大カードで・購入で即開封）＋ **DAILY ARMORY**（武器を行リストで・BUY/SOLD）
- **鍛冶屋（MASTER FORGE）**：対象武器の大パネル（レベル/レアリティ/攻撃力/経験値バー）＋素材選択 or 覚醒 → **強化/覚醒**ボタン
- **インベントリ（ARSENAL）**：所持容量バー＋系統タブ（ALL/斬/突/打）＋武器グリッド（レアリティ・装備・ATK/Lv）。タップで**詳細モーダル**（大画像・ATK/CRIT/BREAK・Modification Log〔スキル〕・装備/ロック/削除）
- **戦闘**：上部に敵、下部に HP/AP セグメントバー → 2×2スキルカード（SLASH/PIERCE/STRIKE/GUARD）→ REST/COMBAT_LOG バー
- **リザルト**：勝敗／到達階・獲得ゴールド・宝箱開封

## ゲームの流れ

1. **ホーム** →「冒険に出る」or 下部ナビで各画面へ
2. ステージを選んで戦闘 → 撃破でゴールド、クリアで武器ドロップ
3. クリアで次ステージ解放。**ショップでゴールド消費**・**インベントリで付け替え/整理**して強化
4. **無限の回廊**で腕試し（最高到達階に挑戦）

## 操作方法

- 武器ボタン（または数字キー `1`〜`3`）：各系統で攻撃（スキルは順番に発動）
- ガード（`Space`）／休憩（`R`）／対象切替（`T`）／敵タップで対象選択

## ディレクトリ構成

```
src/
  game/
    types.ts      # 型定義
    data.ts       # データ読込・各種パラメータ・連携/ショップ/無限の回廊の定義 ※バランス調整はここ
    engine.ts     # 戦闘ロジック（複数敵・ダメージ計算・ガード判定・ブレイク・連携・コイン）
    game.ts       # 全体進行（画面遷移・ウェーブ/無限の回廊・ゴールド・購入・ロック）
    save.ts       # セーブデータ（localStorage）
    weapons.json  # 武器テンプレート
    skills.json   # スキル
    enemies.json  # モンスター
    stages.json   # ステージ
  render/
    canvas.ts     # Canvas描画（戦闘シーン・エフェクト・スプライト→canvasユーティリティ）

  audio/
    audio.ts      # Web AudioによるオリジナルBGM/効果音
  main.ts         # 入力・ゲームループ・画面別UI配線
  style.css
```

## データ編集ガイド（`src/game/*.json`）

表を編集する感覚で敵・武器・スキル・ステージを追加・調整できます。

**enemies.json**（敵）

| フィールド | 内容 |
| --- | --- |
| `id` / `name` | 識別子 / 表示名 |
| `kind` | `carapace`(甲殻) / `phantom`(霊体) / `aerial`(飛翔)。弱点・スプライトを決める |
| `maxHp` / `attack` | 最大HP / 攻撃力 |
| `telegraphMs` | 予兆「!」が出てから着弾までの時間（小さいほどガードが難しい） |
| `countStart` | 攻撃カウントの開始値（プレイヤーの行動ごとに1減る） |
| `breakThreshold` | ブレイクに必要な蓄積量 |
| `dropWeaponId` / `dropRate` | ドロップする武器ID / ドロップ率(%)（任意） |
| `boss` | `true` でボス扱い（大型スプライト）（任意） |

**weapons.json**（武器）

| フィールド | 内容 |
| --- | --- |
| `id` / `name` | 識別子 / 表示名 |
| `weapon` | `slash`(斬) / `pierce`(突) / `crush`(打) |
| `rarity` | `common` / `uncommon` / `rare` / `epic` / `legend` / `astral` |
| `attack` / `critChance` / `breakPower` | 攻撃力 / 会心率(%) / Break蓄積の基礎値 |
| `desc` | 説明文（任意） |

> 新しい武器IDを足したら、`src/render/sprites.ts` の `WEAPON_SPRITES` にドット絵を追加してください（無い場合は系統の絵文字で代替表示）。

**skills.json**（スキル）

| フィールド | 内容 |
| --- | --- |
| `id` / `name` / `rarity` | 識別子 / 表示名 / レアリティ |
| `enCost` | EN消費 |
| `kind` | `attack`(攻撃) / `charge`(ためる) / `focus`(集中) |
| `hits` / `targets` | ヒット数 / 対象数（attack時、既定1） |
| `critAdd` / `critMult` / `breakMult` | 会心率加算(%) / 会心倍率(既定1.5) / Break倍率(既定1) |

**stages.json**（ステージ）

| フィールド | 内容 |
| --- | --- |
| `name` / `desc` | 名称 / 説明 |
| `recommendLv` | 推奨レベル（表示用・任意） |
| `waves` | 各戦闘の敵ID配列の配列（最後がボス戦想定）。例 `[["straw_golem"], ["shell_warden"]]` |
| `endless` | `true` で無限の回廊（`waves`は未使用、敵はランダム生成）（任意） |

その他のテーブルも `src/game/data.ts` 内で編集できます：武器相性 `WEAKNESS` / スキル連携 `COMBOS` / ショップ品揃え `SHOP_ITEMS`・宝箱 `SHOP_CHESTS` / 無限の回廊の敵プール `ENDLESS_POOL`・`ENDLESS_BOSSES` / 武器強化の経験値 `EXP_BASE`・`MATERIAL_EXP`・覚醒 `AWAKEN_STEP`・`MAX_AWAKEN`・成長率 `effectiveWeapon` / 各種バランス定数。
