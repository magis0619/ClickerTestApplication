# ASTRAL WARDEN（仮）

タイミングアクションRPGの **戦闘プロトタイプ**。
「腕次第で格上にも勝てる」テクニカルな戦闘の面白さを核に据えた、完全オリジナルのゲームです。
名称・キャラクター・用語・素材はすべて独自に作成しており、既存作品の固有名詞・アセットは一切使用していません。

## 実装済みの戦闘システム（プロトタイプ）

| システム | 内容 |
| --- | --- |
| **EN（エナジー）** | スキルで消費。**自動回復はしない**。回復は「休憩」と「ガード成功」時のみ（ガード成功の方が大） |
| **武器スキル** | 装備武器ごとに攻撃ボタン1つ。武器固有スキルを**順番に**発動（2つなら交互） |
| **スキル種類** | ⚔単体 / ✸全体(AoE) / ✚HP回復 / ▲ためる（次の攻撃を強化） |
| **武器相性** | 敵の種別（甲殻種・霊体種・飛翔種）ごとに弱点武器があり、弱点ヒットでダメージ増加 |
| **攻撃カウント** | 各敵の頭上に次の攻撃までのカウントを表示。1で敵が震え赤く点滅、予兆「!!」＋警告SEがガードタイミング |
| **タイミングガード** | GUARD（軽減＋EN小・地味）/ **PERFECT**（無効化＋HP/EN大回復＋ヒットストップ＋弾き演出＋敵を怯ませ） |
| **ブレイク（気絶）** | 攻撃でゲージ蓄積→満タンで一定時間ブレイク。ブレイク中は全ヒット会心 |
| **複数敵** | 1ステージに最大3体。タップ/`T`キーで対象選択。最終ステージはボス入り |
| **武器レアリティ** | コモン/アンコモン/レア/エピック/レジェンド/アストラル（虹）。高いほど追加スキルが強く、ドロップで出にくい。撃破時にレアリティ色の宝箱が出現 |
| **インベントリ** | 過去の冒険で入手した武器を格納・付け替え |
| **ドット絵 / サウンド** | コード定義のオリジナルスプライト＋Web Audio合成のBGM・効果音（♪でミュート） |

## ゲームの流れ

1. **タイトル** →「ステージ選択」または「インベントリ」
2. ステージを選んで戦闘 → 勝利でレアリティ抽選の武器がドロップ → インベントリに格納
3. クリアで次ステージ解放。インベントリで強い武器に付け替えて挑戦

## 操作方法

- 武器ボタン（または数字キー `1`〜`3`）：各系統で攻撃（スキルは順番に発動）
- ガード（`Space`）／休憩（`R`）／対象切替（`T`）／敵タップで対象選択

## 開発・実行

技術スタック：TypeScript + HTML5 Canvas + Vite

```bash
npm install
npm run dev      # 開発サーバ（ブラウザで開く）
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果をプレビュー
```

## ディレクトリ構成

```
src/
  game/
    types.ts    # 型定義
    data.ts     # ゲームデータ（スキル・武器・敵・ステージ・各種パラメータ）※バランス調整はここ
    engine.ts   # 戦闘ロジック（複数敵・ダメージ計算・ガード判定・ブレイク）
    game.ts     # 全体進行（タイトル/ステージ選択/インベントリ/戦闘/結果の状態遷移）
    save.ts     # セーブデータ（インベントリ・装備・進行度）の保存/読込（localStorage）
  render/
    canvas.ts   # Canvas描画
    sprites.ts  # ドット絵スプライト定義（オリジナル）
  audio/
    audio.ts    # Web AudioによるオリジナルBGM/効果音
  main.ts       # 入力・ゲームループ・画面別UI配線
  style.css
```
