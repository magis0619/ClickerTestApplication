# ASTRAL WARDEN（仮）

タイミングアクションRPGの **戦闘プロトタイプ**。
「腕次第で格上にも勝てる」テクニカルな戦闘の面白さを核に据えた、完全オリジナルのゲームです。
名称・キャラクター・用語・素材はすべて独自に作成しており、既存作品の固有名詞・アセットは一切使用していません。

## 実装済みの戦闘システム（プロトタイプ）

| システム | 内容 |
| --- | --- |
| **EN（エナジー）** | スキルを撃つたびに消費。時間で自然回復。強い技ほど消費が大きい |
| **スキル** | 武器系統ごとの技をタップして発動（斬撃 / 刺突 / 打撃） |
| **武器相性** | 敵の種別（甲殻種・霊体種・飛翔種）ごとに弱点武器があり、弱点ヒットでダメージ増加 |
| **タイミングガード** | 敵の予兆「!!」に合わせてガード。精度で3段階に分岐 |
| └ GUARD | 被ダメージ軽減 |
| └ JUST GUARD | 大幅軽減 ＋ EN回復 |
| └ PARRY | ダメージ無効化 ＋ HP回復 |
| **ブレイク（気絶）** | 攻撃でブレイクゲージ蓄積→満タンで敵が一定時間ブレイク。ブレイク中は全ヒットが会心＋EN回復 |
| **ダンジョン進行** | 種別の異なる敵が連続で登場。HPを引き継ぎつつ勝利ごとに一部回復。最後はボス戦 |
| **スキル育成** | 撃破で得た「霊片」でスキルを強化（威力・ブレイク蓄積が上昇）。育成内容はブラウザに自動保存され次回も継続 |
| **武器の収集・付け替え** | 敵を倒すと武器をドロップ。系統ごとに装備を切り替えると、その系統スキルの 威力 / 消費EN / ブレイク蓄積 が変化（軽量・高威力・ブレイク特化などの個性） |
| **ドット絵** | プレイヤー・各敵・ボスをコード内定義のオリジナルスプライトで描画 |
| **BGM / 効果音** | Web Audioで合成したオリジナルのチップチューン。右上の「♪」でミュート切替 |

## ゲームの流れ

1. ステージの敵を撃破 → 「STAGE CLEAR」で霊片を獲得
2. 報酬画面でスキルを強化（任意）→ 次のステージへ
3. 全ステージ撃破で「DUNGEON CLEAR」。敗北しても育成は保持され、再挑戦で強くなって挑める

## 操作方法

- スキルボタン（または数字キー `1`〜`6`）：攻撃
- ガードボタン（または `Space`）：敵の予兆に合わせて防御
- 報酬画面：「次のステージへ」（`N`）
- 敗北／クリア画面：「再挑戦」（`R`）

## 開発・実行

技術スタック：TypeScript + HTML5 Canvas + Vite

```bash
npm install
npm run dev      # 開発サーバ（ブラウザで開く）
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果をプレビュー
```

## ディレクトリ構成

```
src/
  game/
    types.ts    # 型定義
    data.ts     # ゲームデータ（スキル・武器・敵・ステージ・各種パラメータ）※バランス調整はここ
    engine.ts   # 戦闘ロジック（ダメージ計算・ガード判定・ブレイク）
    game.ts     # 全体進行（ダンジョン・報酬・育成・武器装備・敗北の状態遷移）
    save.ts     # セーブデータ（育成・所持武器・装備）の保存/読込（localStorage）
  render/
    canvas.ts   # Canvas描画
    sprites.ts  # ドット絵スプライト定義（オリジナル）
  audio/
    audio.ts    # Web AudioによるオリジナルBGM/効果音
  main.ts       # 入力・ゲームループ・画面別UI配線
  style.css
```
