# ASTRAL WARDEN（仮）

タイミングアクションRPGの **戦闘プロトタイプ**。
「腕次第で格上にも勝てる」テクニカルな戦闘の面白さを核に据えた、完全オリジナルのゲームです。
名称・キャラクター・用語・素材はすべて独自に作成しており、既存作品の固有名詞・アセットは一切使用していません。

## 実装済みの戦闘システム（プロトタイプ）

| システム | 内容 |
| --- | --- |
| **EN（エナジー）** | スキルを撃つたびに消費。時間で自然回復。強い技ほど消費が大きい |
| **スキル** | 武器系統ごとの技をタップして発動（斬撃 / 刺突 / 打撃） |
| **武器相性** | 敵の種別（甲殻種・霊体種・飛翔種）ごとに弱点武器があり、弱点ヒットでダメージ増加 |
| **タイミングガード** | 敵の予兆「!!」に合わせてガード。精度で3段階に分岐 |
| └ GUARD | 被ダメージ軽減 |
| └ JUST GUARD | 大幅軽減 ＋ EN回復 |
| └ PARRY | ダメージ無効化 ＋ HP回復 |
| **ブレイク（気絶）** | 攻撃でブレイクゲージ蓄積→満タンで敵が一定時間ブレイク。ブレイク中は全ヒットが会心＋EN回復 |

## 操作方法

- スキルボタン（または数字キー `1`〜`6`）：攻撃
- ガードボタン（または `Space`）：敵の予兆に合わせて防御
- もう一度ボタン（または `r`）：再戦

## 開発・実行

技術スタック：TypeScript + HTML5 Canvas + Vite

```bash
npm install
npm run dev      # 開発サーバ（ブラウザで開く）
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果をプレビュー
```

## ディレクトリ構成

```
src/
  game/
    types.ts    # 型定義
    data.ts     # ゲームデータ（スキル・敵・各種パラメータ）※バランス調整はここ
    engine.ts   # 戦闘ロジック（ダメージ計算・ガード判定・ブレイク）
  render/
    canvas.ts   # Canvas描画
  main.ts       # 入力・ゲームループ・UI配線
  style.css
```
